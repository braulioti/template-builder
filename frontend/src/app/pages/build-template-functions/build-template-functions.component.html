Functions

Functions allow you to transform variable values inside placeholders. You use them by writing the function name followed by the argument in parentheses inside double curly braces (e.g. {{ functionPlaceholder }}). The Template Builder evaluates the function and replaces the placeholder with the result. Functions can be used in file content and in prompt results. Functions can also be nested (e.g. {{ nestedExample }}).

Available Functions

Name Arguments Description
upper 1: variable or string Converts the argument to uppercase. Example: {{ upperExample }} converts the value of variableName to all uppercase letters.
lower 1: variable or string Converts the argument to lowercase. Example: {{ lowerExample }} converts the value of variableName to all lowercase letters.
replace 3: search, replace, variable or string Replaces all occurrences of the first argument (search) with the second argument (replace) in the third argument (variable or string). Example: {{ replaceExample }} replaces all spaces with underscores in the value of variableName.

Chained Functions

Functions can be chained: the result of one function is passed as the argument to another. The Template Builder evaluates the innermost function first, then the next, and so on. For example, {{ nestedExample }} first replaces spaces with underscores in the variable value, then converts the result to lowercase. Chaining lets you combine transformations (e.g. normalize a name for use as an identifier or filename).
