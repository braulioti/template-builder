Build your first template

The template is defined in a YAML file. In it you declare folders, files, variables, and prompts. Template Builder reads this file and generates the project structure in the current directory.

Every template project must specify the format version at the top of the YAML. This version must match the Template Builder version without the third number: for example, for Template Builder 0.1.0, use version: 0.1 in the file.

Example: folder and file

Create a file named hello.yaml with the content below. It defines a folder New Folder and a file Hello.txt inside it with the content Hello World.

version: 0.1

folders:
  - path: "New Folder"

files:
  - path: "New Folder/Hello.txt"
    content: "Hello World"

To generate the folder and file, run the following in a terminal, in the same directory as hello.yaml:

TemplateBuilder hello.yaml
