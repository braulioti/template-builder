Remote Files

Remote files let your template download content from URLs (e.g. scripts, configs, or assets) and place them into the generated project. You declare each remote file in the template YAML under the remote key, with uri (the URL to fetch) and path (where to save it). The Template Builder fetches the file and writes it at the given path when the template is run.

Attributes

Name Required Description
uri required URL of the file to download. Must be a direct link to the resource (e.g. https://example.com/file.svg), not a wiki or HTML page that displays the file.
path required Destination path where the downloaded file will be saved in the generated project. Relative to the project output directory (e.g. assets/HelloWorld.svg, languages/compile-mo.php).
