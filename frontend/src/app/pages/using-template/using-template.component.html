Using a template

This guide shows how to run a ready-made template: download it, save it in a folder, and run Template Builder from the command line.

Step 1 — Download a template

On the Installation page, open the Samples section. Download one of the sample templates (for example, readme.yaml) to your computer.

Step 2 — Save the template in a folder

Create a folder where you want to use the template (for example, my-project or C:\templates on Windows). Save the downloaded .yaml file in that folder. You can also place it in any existing folder where you have write permissions.

Step 3 — Run the template from the command line

Open a terminal in the folder where Template Builder is installed (or ensure it is on your PATH). Run the executable passing the path to your YAML file.

TemplateBuilder path/to/your/template.yaml

Template Builder will parse the YAML, run any prompts (project name, version, etc.), and generate the files and folders defined in the template.
